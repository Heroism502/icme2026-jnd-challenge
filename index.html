ICME 2026 Grand Challenge

Perceptual Just Noticeable Difference (JND) Modeling for 3D Point Cloud Compression

Introduction

Point clouds are a fundamental 3D representation widely used in immersive media, autonomous driving, digital twins, AR/VR, and telepresence. Compression is unavoidable in real systems and can cause perceptual quality degradation.

Just Noticeable Difference (JND) characterizes the minimum distortion level at which quality degradation becomes perceptible. Accurate JND modeling supports perceptually optimized compression.

Dataset: PKU-JND

  • 230 reference point cloud models across 5 content categories
  • 5,750 geometry-distorted samples and 7,130 attribute-distorted samples
  • Generated using MPEG G-PCC
  • Large-scale subjective experiments with rigorous post-processing
Dataset download links will be released with the official challenge announcement.

Tasks

  • Geometry JND prediction
  • Attribute (color) JND prediction
  • Unified perceptual JND modeling

Evaluation

  • PLCC, SRCC, KRCC, RMSE

Final leaderboard score: $$\text{Score} = \frac{\text{PLCC} + \text{SRCC}}{2}$$

Only the official training data may be used. External data usage is strictly prohibited.

Submission

Submit predicted JND values for the official test set in the provided JSON format. Up to three submissions per team per day.
