Perceptual JND Prediction for 3D Point Cloud Compression

Introduction

Point clouds are a core 3D representation in immersive media, autonomous vehicles, telepresence, AR/VR, and digital twins. Compression is essential in real systems, but must preserve perceptual quality.

Just Noticeable Difference (JND) measures the minimal perceptual distortion level. Accurate JND modeling enables perceptually optimized compression that removes imperceptible details while keeping visually important structures.

Dataset — PKU-JND

  • 230 point cloud models across 5 categories
  • 5,750 geometry-distorted samples
  • 7,130 attribute-distorted samples
  • Generated via MPEG G-PCC
  • Subjective experiments with rigorous post-processing

Dataset links will be announced.

Evaluation

  • PLCC — Pearson Linear Correlation Coefficient
  • SRCC — Spearman Rank-Order Correlation
  • KRCC — Kendall Rank Correlation
  • RMSE — Root Mean Squared Error

Final leaderboard score:

$$\text{Score} = \frac{\text{PLCC} + \text{SRCC}}{2}$$
